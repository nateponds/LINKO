# LINKO hero product walkthrough

Place the final recorded product-session video at:

```text
public/videos/linko-product-tour.mp4
```

The landing-page hero will detect the file automatically and replace the
placeholder with the browser's native video player.

Recommended delivery:

- 16:9 aspect ratio
- 1920×1080 or 1280×720
- H.264 MP4 with web-optimized streaming enabled
- Demonstrate supplier discovery, bulk-price comparison, order management,
  and parcel tracking
- Avoid real customer data, credentials, browser bookmarks, or notifications
